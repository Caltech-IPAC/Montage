Test Suite Overview

The Montage project has carried out a thorough set of functional, system and verification (acceptance) testing. These test results will be made available to interested parties who request them by email (montage@ipac.caltech.edu).

Build Test: Once you have installed Montage, you can run a "build test" (actually a shell script depending on a number of small C programs). This script runs the Montage code on a synthetic image and compares the results to the original image in order to show that Montage is installed properly.
